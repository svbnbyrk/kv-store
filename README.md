## Getting Started

kv-store is basically REST api that works like a key-value store.

### Installation & Usage

1. Clone the repo

   ```sh
   git clone https://github.com/svbnbyrk/kv-store.git
   ```

2. Go inside project

   ```sh
   cd kv-store
   ```

3. Build docker file

   ```sh
   docker build -t kv-store .   
    ```

4. Run container

   ```sh
   docker run -d -p 9090 kv-store  
    ```

Ready to go!
